# CS50 - Week 0 (Scratch) — Dodge Game

## Sobre
Projeto do Módulo 0 do CS50 feito no Scratch.  
Objetivo: desviar do obstáculo e sobreviver o máximo possível.

## Como jogar
- Use as setas **← →** para mover o personagem.
- Se encostar no obstáculo: **GAME OVER**.

## Requisitos do CS50 atendidos
- [x] Pelo menos **2 sprites** (1 não é gato)
- [x] Pelo menos **3 scripts**
- [x] Usa **condição** (if)
- [x] Usa **loop** (forever)
- [x] Usa **variável** (`score`)
- [x] Usa **som** (game over)

## O que eu implementei
- Movimento horizontal do personagem com teclado
- Obstáculo com **posição aleatória no eixo X**
- Pontuação que aumenta com o tempo
- Detecção de colisão + som + finalização do jogo

## O que eu aprendi
- Eventos (`when green flag clicked`) e loop (`forever`)
- Variáveis e atualização de estado (`score`)
- Colisão (`touching`) e controle de fluxo (`if`)
- Aleatoriedade (`pick random`) pra variar a jogabilidade
